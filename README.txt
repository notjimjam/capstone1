Capstone Project- Puzzling

I created a site for all the puzzle obsessed people out there, like myself.
It will allow you to view puzzles that other have completed and post
your own puzzling fun. Share your puzzle journey with other like you through
images, links, and helpful tips.

Features:
- View an update-to-date post feed
- Click each post for an expanded view, including descriptions
- Create new posts to share the things you've built, which gets stored into the database

Tech Stack
- Javascript
- HTML
- CSS
- Express
- Postgres
- Node.js
